# Decoding Cyber Threats and Digital Trust through Risk Analytics

## Overview

This project is a Python-based desktop application developed to identify potential cyber threats by analyzing employee login behavior and activity patterns using risk analytics techniques. The system detects suspicious activities such as repeated failed login attempts, unusual login timings, and unfamiliar device access to improve cyber security monitoring and strengthen digital trust.

The project also integrates Power BI dashboards for interactive visualization and threat analysis.


## Features

* Employee login activity monitoring
* Cyber threat detection and analysis
* Risk score calculation
* Risk level classification
* Suspicious login identification
* Interactive Power BI dashboard
* Threat logs and monitoring
* Email alert support
* User authentication system

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* SQLite
* Power BI
